# Lodelix

A rust-written web applications server.

## Features

- Configuration via RESTful API
- Configuration via gRPC
- Configuration via AI agents

## Build

### Development

1. `cargo build`
2. `cargo run`

### Production

1. `cargo build --release`
2. `cargo run --release`
